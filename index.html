《权力的游戏：走向共和》

发表于 2019-05-21 | 分类于 杂记

权力的游戏彻底完结了，实话说，最终季确实崩了。但我觉得网上的大多不满有些不太恰当。首先原著作者马丁多年前已经把大结局的主要人物结局透露给了两位制作人（不出意外，原著也可能会这样写），然后，按照前几集的铺垫，结局的可能性已经被收的很窄了，你说龙妈如果不死是不是会招致更大的不满呢？所以权游的结局本身是还算合理的，至少在我来看是这样。最终季真正崩的并不是结局，而是迈向结局的一整个过程以及其中所有的细节。说实话，你让两位制作人在没有原著的指导下凭空创造出和媲美原著的精彩故事基本上是很难的（马丁可是写了20多年啊），但真正可惜的是，最终季甚至连及格线都没达到！整个最终季让观众感受到两个制作人急于彻底结尾的粗糙感（难不成是等不及去祸害更能骗钱的星战新三部曲？）。

先说结局本身。要知道马丁是个重度历史迷。权游的原著《冰与火之歌》也和很多历史事件吻合。大结局很符合马丁全书的气质。龙逐渐变少直到彻底消亡，低魔过渡到无魔世界。看似正统的两个坦格利安继承人全都没有走上王卫，旧世界唯血统论的世袭制彻底终结，领袖改由领主选举，甚至还埋下了人民直接选举的种子。这不就是走向共和的局势吗。我反正觉得这个结局本身挺好的。

然后才是万人吐槽的剧情推进，首先，人物的转变跟闹着玩一样，总感觉所有人都变成了”愣头青”，突然一条路走到黑的样子。龙妈莫名其妙变成了灭霸人设，开始宣扬消灭旧世界创造新世界的无聊理念。最终集里你会发现雪诺从一开始扮演了一个超级唯唯诺诺的怂货，直到他吻里藏刀的那一刻，你都不清楚他是从哪一刻改变了自己的想法的，二脸懵逼。最终，好多人物都成了精神分裂患者。其实龙母与雪诺二人之间的故事完全可以处理成一场盛大的悲剧的，这也恰恰是大结局的最大看点。可惜编剧已经没心思去花时间去铺垫故事细节了，6集的长度（虽然很多集很长）完全不能承担剧情的展开。

紫霞曾经对至尊宝说：我只猜中了前头，可是我猜不着这结局！

编剧：我提前知道了结局，但是没能力甚至也不想花力气去推进整个过程！

请以你的名字呼唤我

发表于 2017-12-30 | 分类于 电影

分享一段电影中最后父亲的独白，我觉得这段话适用于世间任何的情感或者说体验？我在想是不是只有真正经历过遗憾的人才会有父亲这样的感受，而正在当下的人永远无法感同身受？或者说当你已然看透的时候其实你已经变得麻木了？

“在你最想不到的时候，命运会以最狡猾的方式找到我们的软肋。但你要记住，我在这儿。现在你也许希望自己不去感觉，也许你希望从不感觉这些，也许你是不想跟我说这些事情，但是…去感觉这些明显的感觉吧！你有过一段美妙的友谊，也许比友谊更多，这让我羡慕。站在我的角度，多数父母会希望整件事淡去，祈祷他们的孩子重新振作 但…我不是那样的家长。为了让我们更快愈合，我们从自己身上剥去了太多东西，以致于不到30岁就一无所有。每开始一段新的感情，能给予对方的就越少。但为了让自己没有感觉而不去感觉，这是多么的浪费! ”

“我说了不该说的话吗？ 那我再说一件事情，让我们扫除芥蒂。我可能曾经很接近，但我从没有过你们所拥有的。总有什么，牵绊着我，挡在我的前面。怎么过这一生是你的事情。只要…记住，我们的心和身体只被给予这一次，而在你知道前，你的心会疲惫，而你的身体…总有一天没人会愿意多看一眼，更不要说贴近它。现在…你可能感到难过，痛苦，别让它消失，更别说那些有过的快乐。”

般若波罗蜜多心经

发表于 2017-12-05 | 分类于 杂记

观自在菩萨，行深般若波罗蜜多时，照见五蕴皆空，度一切苦厄。舍利子，色不异空，空不异色；色即是空，空即是色。受、想、行、识，亦复如是。舍利子，是诸法空相，不生不灭，不垢不净，不增不减，是故空中无色，无受、想、行、识；无眼、耳、鼻、舌、身、意；无色、声、香、味、触、法；无眼界，乃至无意识界；无无明，亦无无明尽；乃至无老死，亦无老死尽。无苦、集、灭、道，无智亦无得。以无所得故，菩提萨埵，依般若波罗蜜多故，心无罣碍，无罣碍故，无有恐怖，远离颠倒梦想，究竟涅槃。三世诸佛，依般若波罗蜜多故，得阿耨多罗三藐三菩提。故知般若波罗蜜多，是大神咒，是大明咒，是无上咒，是无等等咒，能除一切苦，真实不虚。故说般若波罗蜜多咒，即说咒曰：“揭谛、揭谛，波罗揭谛，波罗僧揭谛，菩提萨婆诃。”
— 玄奘法师译，般若波罗蜜多心经

《但丁密码》:十年之后，我再次睡着

发表于 2016-11-02 | 分类于 电影
